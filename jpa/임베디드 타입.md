# 임베디드 타입

### Value Class
- 여러 값이 모여져서 하나의 값을 나타내는 클래스
    ```java
    class Address{
        String city;
        String street;
        String detail;
        String zipCode;
    }
    ```

### @Embeddable @Embedded
- 설정한 클래스가 다른 엔티티의 일부로 저장될 수 있음을 설정
    ```java
    class User{
        @Id
        private Long id;
        
        private String name;

        @Embedded
        private Address address;
    }
    ```

    ```java
    @Embeddable
    class Address{
        private String city;
        private String street;
        private String detail;
        private String zipCode;
    }
    ```
